# Instagram Anonymous Story Viewer (IASV)

Anonymous story viewing for Instagram.

## What It Does
- Blocks Instagram story-seen signals so your views stay private.
- Works on Chrome, Edge, and Firefox.

## Privacy
IASV does not collect, store, or transmit personal data. It only blocks specific network requests on `instagram.com`.
